# 한국어 어순
총 여섯 개가 있다.

* 주어 + 서술어
  * '주어'는 '서술'이다.
  * `( <주어>* <서술어> )`
    * `(3 출력)`, `(2 3 더하기)`

* 주어 + 보어 + 서술어
  * '주어'는 '보어'가 '서술'이다.
  * `( {<주어>* <목적어>*}* <서술어> )`
    * `(여기 서울 동일하다)`

* 주어 + 목적어 + 서술어
  * '주어'는 '목적어'를 '서술'한다.
  * `( {<주어>* <목적어>*}* <서술어> )`

* 주어 + 부사어 + 서술어
  * '주어'는 '부사어'인 '서술'이다.

* 주어 + 부사어 + 목적어 + 서술어
  * '주어'는 '부사어'인 '목적어'를 '서술'한다.

* 주어 + 목적어 + 부사어 + 서술어
  * '주어'는 '목적어'를 '부사어'인 '서술'이다.
